== Description ==

Include Rentivo widgets on your website, including real time availability and book now functionality.

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/rentivo-widgets` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Use the Rentivo Widgets menu item in the left sidebar to configure the plugin
4. Click settings on the Rentivo widget you'd like to configure and follow onscreen instructions
